# Angular Learning Guide

Welcome to the Angular documentation! This guide covers the essential concepts and features you need to build scalable, maintainable web applications with Angular, a powerful TypeScript-based framework.

## What is Angular?

Angular is a comprehensive, open-source framework for building dynamic web applications. It provides a complete solution for frontend development with built-in tools for routing, form handling, HTTP communication, and state management. Angular uses TypeScript and reactive programming principles to help you build performant, testable applications.

## Key Concepts Covered

This documentation covers:

- **AI Tutor Setup** - Setting up the Angular AI tutor for interactive learning with the Gemini CLI and smart guided tutorials
- **Core Fundamentals** - Components, templates, directives, and services
- **Dependency Injection** - Angular's powerful dependency injection system
- **Routing** - Building multi-page applications with Angular Router
- **Forms** - Handling user input with template-driven and reactive forms
- **HTTP Communication** - Making API requests and managing data
- **TypeScript** - Modern JavaScript with type safety
- **Testing** - Unit testing and end-to-end testing strategies

## Prerequisites

You should have basic knowledge of:
- JavaScript (ES6+ syntax preferred)
- HTML and CSS
- TypeScript basics or willingness to learn
- Node.js and npm/pnpm
- Command line/terminal usage

## Getting Started

To get started with Angular, install Node.js and create a new Angular project:

```bash
npm install -g @angular/cli
ng new my-angular-app
cd my-angular-app
ng serve
```

Your application will be available at `http://localhost:4200/`.
