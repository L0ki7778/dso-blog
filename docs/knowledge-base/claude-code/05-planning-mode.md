# Planning Mode und Claude Design

- Der **Planner Mode** lässt sich per Prompt oder mit `SHIFT + TAB`
  aktivieren.
- Unter [claude.ai/design](https://claude.ai/design) lässt sich ein Design
  exportieren und direkt in den Context ziehen. Auch ein einfacher
  Screenshot funktioniert dafür.

:::tip
Für Planning-Sessions empfiehlt es sich, ein exportiertes Design oder einen
Screenshot direkt mit in den Context zu geben.
:::
