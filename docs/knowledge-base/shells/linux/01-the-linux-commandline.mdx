# Linux Commands (Ubuntu)

This page provides a structured overview of the most important Linux commands you can use in your daily Ubuntu work.

---

## 📁 File and Directory Management

| Command | Description |
| ----------- | ----------- |
| `ls`|	Lists files and directories |
| `cd <directory>`|	Changes the directory |
| `pwd`|	Shows current directory |
| `mkdir <name>`|	Creates a new directory |
| `rmdir <name>`|	Removes empty directory |
| `rm <file>`|	Deletes file |
| `cp <source> <target>`|	Copies files/folders |
| `mv <source> <target>`|	Moves or renames |

### Flags & Details

<details>
<summary><strong>ls</strong> — Lists files and directories</summary>

* `ls -l` → Detailed view
  * Example: `ls -l` → shows file sizes, permissions, owner

* `ls -a` → Show hidden files
  * Example: `ls -a` → shows `.git`, `.env`

* `ls -lh` → Readable sizes
  * Example: `ls -lh` → 1K, 234M instead of bytes

</details>

<details>
<summary><strong>cd</strong> — Changes directory</summary>

* `cd ..` → Go one level up
  * Example: `cd ..` → from `/home/user/docs` to `/home/user`

* `cd ~` → Home directory
  * Example: `cd ~` → `/home/user`

* `cd -` → Previous directory
  * Example: `cd -` → back to previous folder

</details>

<details>
<summary><strong>mkdir</strong> — Creates directories</summary>

* `mkdir -p` → Creates nested folders
  * Example: `mkdir -p new_folder/demo` → Path: `/home/user/new_folder/demo`

</details>

<details>
<summary><strong>rm</strong> — Deletes files and folders</summary>

* `rm -r` → Recursive
  * Example: `rm -r folder` → deletes entire folder

* `rm -f` → Force
  * Example: `rm -f file.txt` → delete without confirmation

</details>

<details>
<summary><strong>cp</strong> — Copies files</summary>

* `cp -r` → Copy folder
  * Example: `cp -r src backup` → copies entire folder

</details>

<details>
<summary><strong>mv</strong> — Moves or renames</summary>

* `mv file.txt new.txt` → Rename
  * Path: `/home/user/new.txt`

* `mv file.txt folder/` → Move
  * Path: `/home/user/folder/file.txt`

</details>

## 📄 View and Edit File Contents

| Command | Description |
| ----------- | ----------- |
|`cat <file>`	| Shows entire file content|
|`less <file>`	| Scrollable view|
|`head <file>`	| Show first lines|
|`tail <file>`	| Show last lines|
|`nano <file>`	| Simple editor|
|`vim <file>`	| Advanced editor|

### Flags & Details

<details>
<summary><strong>cat</strong> — Shows file content</summary>

* `cat -n` → Line numbers
  * Example: `cat -n file.txt`

</details>

<details>
<summary><strong>less</strong> — Scrollable view</summary>

* `less file.txt` → Scroll with arrow keys

</details>

<details>
<summary><strong>tail</strong> — Show last lines</summary>

* `tail -f` → Live logs
  * Example: `tail -f app.log`

</details>

<details>
<summary><strong>head</strong> — Show first lines</summary>

* `head <file>` → First 10 lines
  * Example: `head file.txt`

* `head -n 20 <file>` → First 20 lines
  * Example: `head -n 20 file.txt`

</details>
