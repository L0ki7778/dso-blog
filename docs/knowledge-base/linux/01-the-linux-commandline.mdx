Linux Befehle für Ubuntu
Diese Seite bietet eine strukturierte Übersicht über die wichtigsten Linux-Befehle, die du im Alltag unter Ubuntu verwenden kannst.
---


📁 Datei- und Verzeichnisverwaltung
| Befehl | Beschreibung |
| ----------- | ----------- |
| `ls`|	Listet Dateien und Verzeichnisse auf |
| `ls -la`|	Detaillierte Ansicht inkl. versteckter Dateien |
| `cd <verzeichnis>`|	Wechselt das Verzeichnis |
| `pwd`|	Zeigt aktuelles Verzeichnis |
| `mkdir <name>`|	Erstellt ein neues Verzeichnis |
| `rmdir <name>`|	Löscht leeres Verzeichnis |
| `rm <datei>`|	Löscht Datei |
| `rm -r <ordner>`|	Löscht Ordner rekursiv |
| `cp <quelle> <ziel>`|	Kopiert Dateien/Ordner |
| `mv <quelle> <ziel>`|	Verschiebt oder benennt um |
